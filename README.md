# MVP FrontEnd UNIFESO 
                                              Circuito Terê Verde

## 

## Sobre o projeto:

- Repositório criado para a elaboração do MVP FrontEnd do curso de Análise e desenvolvimento de sistemas (ADS).
- Este MVP de modelo protótipo tem a finalidade de promover o site "Circuito Terê Verde", a fim de facilitar aos moradores e visitantes da cidade de Teresópolis a pesquisa de informações. 
- O site proverá sobre os parques Três picos, Serra dos Orgãos e Montanhas de Teresópolis, informações sobre trilhas, programações e demais dados sobre suas biodiversidades de fauna e flora.
- Para acessar o site: [Clickable Text](https://mateusty.github.io/MVP-FrontEnd)
