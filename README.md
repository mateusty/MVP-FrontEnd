# MVP FrontEnd UNIFESO 
                                                    Circuito Terê Verde
 
## Sobre o projeto:

- Repositório criado para a elaboração do mvp do 4° trimestre do curso de Análise e desenvolvimento de sistemas(ADS).
- Este MVP de modelo protótipo tem a finalidade de promover o site "circuito terê", a fim de promover facilidade aos moradores e visitantes da cidade de teresópolis. 
- O site circuito terê proverá sobre os parques _Três picos_, Serra dos Orgãos,e Montanhas de Teresópolis, informações sobre trilhas, programações e demais dados sobre suas biodiversidades de fauna e flora.
